# chatsdk_quick_start

A quick start sample project for Easemob (Agora) Flutter Chat SDK, demonstrating basic one-to-one messaging.

## Features

- SDK initialization
- User sign in / sign out
- Send text messages
- Receive messages

## Getting Started

1. Create an app in the [Agora Console](https://console.agora.io/) and get the App ID
2. Create a user and obtain a Token
3. Replace the `AgoraChatConfig` in `lib/main.dart` with your credentials
4. Run the project:

```bash
flutter pub get
flutter run
```
